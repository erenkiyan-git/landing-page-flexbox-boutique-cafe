# The Cozy Bean | Boutique Cafe Landing Page ☕

A modern, clean, and responsive landing page designed for a fictional boutique cafe. This project was built to demonstrate proficiency in semantic HTML5 and advanced CSS3 layouts, with a heavy emphasis on **Flexbox** architecture.

## 📖 About The Project

This landing page is designed to capture the warm, inviting atmosphere of an artisan cafe while maintaining a strict, professional web layout. It features a hero section, an informational product grid, a customer testimonial, and a call-to-action (CTA) block. 

The core challenge and focus of this project was to completely avoid absolute positioning or floats, relying entirely on modern CSS Flexbox principles to align, distribute, and structure the content across the page.

### Key Features
* **Semantic HTML5:** Proper use of `<header>`, `<section>`, `<nav>`, and `<footer>` tags for accessibility and SEO.
* **CSS Flexbox Mastery:** Extensive use of nested flex containers, `justify-content`, `align-items`, and `gap` properties for flawless component alignment.
* **Clean UI/UX:** A carefully selected color palette (Dark Slate, Clean White, and Action Blue) with Roboto typography for maximum readability.
* **Image Handling:** Utilization of `object-fit: cover` to maintain image aspect ratios within strictly sized placeholder cards.

## 🛠️ Built With

* HTML5
* CSS3 (Vanilla)
* Flexbox
